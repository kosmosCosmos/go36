#go语言里面的引用类型和值类型#


***

在go语言里面有三种引用类型，slice（切片），map(字典)，channel（管道）

	func Test1(){
		a := 3
		b := a
		b = 100
		fmt.Println(&b,&a,b,a) //b相当于copy生成的，和a指向不同的内存
	}
	func Test2(){
		s1 := []int{1,2,3,4}
		s2 := s1 				//s2和s1是相同的内存的
		s2[0] = 8			//这里改变了s2的话，s1也会更改
		fmt.Println(s1,s2)
	}

	0xc042054088 0xc042054080 100 3

	[8 2 3 4] [8 2 3 4]

**go语言里面不存在深拷贝，对于值类型，go都是统一copy过来的。（看起来像深拷贝。）**
